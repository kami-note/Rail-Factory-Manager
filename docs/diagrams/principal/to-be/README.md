# Use Case Diagrams - TO-BE

Este diretório contém os diagramas de caso de uso no cenário **to-be**, separados por domínio.

## Arquivos

- `geral-use-case-to-be-v1.puml` - Visão geral com todos os domínios.
- `iam-use-case-to-be-v1.puml` - Domínio IAM.
- `production-use-case-to-be-v1.puml` - Domínio Production.
- `supply-chain-use-case-to-be-v1.puml` - Domínio Supply Chain.
- `logistics-use-case-to-be-v1.puml` - Domínio Logistics.
- `fleet-use-case-to-be-v1.puml` - Domínio Fleet.
- `hr-use-case-to-be-v1.puml` - Domínio HR.
- `dashboard-use-case-to-be-v1.puml` - Domínio Dashboard & Reporting.

## Renderização

Na raiz do repositório:

```bash
./scripts/render-plantuml-diagrams.sh
```

Como os arquivos seguem o padrão `*-v*.puml`, eles também serão renderizados automaticamente.
