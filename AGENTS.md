# AI Agent Guidelines

Este projeto utiliza o **GEMINI.md** (na raiz) como a fonte única de verdade para mandatos de engenharia, protocolos elite e padrões de código.

## 📌 Links Importantes
- **[GEMINI.md](./GEMINI.md)**: Mandatos de Engenharia, Protocolos Elite e Regras de Qualidade.
- **[docs/CONTEXTO_ATUAL.md](./docs/CONTEXTO_ATUAL.md)**: Estado atual do projeto e próxima tarefa.
- **[docs/PLANO_DE_TASKS.md](./docs/PLANO_DE_TASKS.md)**: Backlog detalhado.

## 🚀 Como trabalhar neste projeto
1. **Leia o `GEMINI.md`** para entender os protocolos de prevenção de retrabalho.
2. **Consulte o `docs/CONTEXTO_ATUAL.md`** para saber onde o sistema parou.
3. **Verifique o `docs/PLANO_DE_TASKS.md`** para a sua tarefa ativa.
4. **Siga a Arquitetura Hexagonal**: Domínio isolado, Application para orquestração, Infrastructure para persistência.
5. **Documente com XML/TSDoc**: Foque na intenção e nas invariantes.
